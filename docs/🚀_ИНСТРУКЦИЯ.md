## ✅ ПРОВЕРКА:

После деплоя проверь:

```bash
# На сервере
pm2 status
```

Должно быть:
```
┌────┬────────────────────────┬─────────┬─────────┬──────────┐
│ id │ name                   │ mode    │ ↺       │ status   │
├────┼────────────────────────┼─────────┼─────────┼──────────┤
│ 0  │ nubo-training-frontend │ fork    │ 0       │ online   │
│ 1  │ nubo-training-backend  │ fork    │ 0       │ online   │
└────┴────────────────────────┴─────────┴─────────┴──────────┘
```

**Оба `online` и `↺ 0` - значит все работает!** 🎉
